再论 “因为” 词类

发表于 2023-07-13 分类于 作业 ， 现汉 阅读次数：
本文字数： 5.8k 阅读时长 ≈ 5 分钟

摘要：本文从学界对于 “因为” 词类划分问题的分歧出发，讨论 “因为” 一词的多种语法分布情况，兼而讨论连词和介词的判断标准。“因为” 的词类是在一连续统上进行变化的。“因为 + np” 是典型的介宾结构。“因为（+np）+vp” 逐渐由介词向连词转化。当 “因为” 后接的主谓结构具有了句子的特点，“因为” 就已经完全成为连词。

关键词：“因为” 词类 连词 介词 复句

再论 “因为” 词类

一、“因为” 词类划分的分歧

自《马氏文通》起，学界对连词的划分问题一直有所讨论。《马氏文通》中分出连字、介字、状字，但无法界限分明地划定三者的区别，导致他在分析过程时常有纠缠，标准不一。赵元任也指出 “中文里的连接词跟介词、副词很难分得开…… 事实上中文里连接词的性质很难确定，所以龙索夫索性不把他当作独立的一个词类来看。”[①] 因为语法分布的相似和兼类词的存在，连词与介词、副词在某些情况难以区分。

例如，《现代汉语八百词》中的 “因为” 分为两种词性：介词、连词。本书并未明确指出两种词性的判断依据，但是根据其阐释和例句可以大体推断出，介词的 “因为” 连接的是体词性成分，连词的 “因为” 连接的是谓词性成分。《现代汉语词典》、沈阳和郭锐的《现代汉语》也列出了 “因为” 的两种词性，分类标准和《八百词》大体一致。但是，宋绍年、章欣提出，要对 “因为” 一词的词性进行重新思考，他们认为 “因为” 可以作为介词连接谓词性成分，并且对汉语偏正复句的合理性提出质疑 [②][③]。

词类划分主要是以语法分布及语法功能为依据的，要考察 “因为” 的词类，首先要分析 “因为” 在语境中的语法分布。本文对 BCC 及 CCL 语料库中包含 “因为” 的现代汉语语料进行句法结构分析，提取 “因为” 所在结构的主干成分，得到以下三种句法结构模式：

语法结构 例句
因为 + np 但因为固执于地方自治学说的缘故，《宪草》所规定是十分凌杂的。
因为 + vp 山东有个中学有些学生因为过分用功，经常头痛。
因为 + np+vp 厂领导因为我无故矿工，给我记大过处分，并且把我赶出了工厂宿舍。

其中，“因为 + np” 模式没有争议，学界普遍认为这一结构中的 “因为” 是介词，因为介词连接体词性短语是其常规语法功能，而偏正复句中的连词不具有这项功能。第二种情况 “因为 + vp”，陆俭明在论述表示主从关系的连词时曾提出 “在表示主从关系的连词中，有少部分能用来连接主语和谓语。”[④] 那么 “因为 + vp” 应是 “因为 + np+vp” 的一种简略形式，属于连词 + 分句的一种特殊情况。第三种情况 “因为 + np+vp” 已经有了较为完整的主谓结构，大多数学者都将之视为一个分句，从而认为 “因为” 是连接分句与分句的连词。而章欣则认为，由于三种 “因为” 结构后接成分皆为指称性成分，并无本质区别，所以后二种情况的 “因为” 皆属介词。

可以看出，关于 “因为” 的词类划分的分歧主要集中于 “因为 + vp” 和 “因为 + np+vp” 结构，下面主要讨论这两种结构中的 “因为” 词类。

二、“因为（+np）+vp” 与 “介词 + 谓宾” 同理

介词将自己的宾语引介给谓语，整个介宾词组在句中充当状语。连词能够连接词或词以上的语言单位，表示各种逻辑关系。在复句中，连词主要用于分句与分句的连接。“因为（+np）+vp” 属连词的观点是将 “因为 + np” 的模式与 “因为（+np）+vp” 分开，认为介词不能连接谓词性短语从而获得 “因为” 作为连词的合理性。然而，这种分类方法存在局限，“因为（+np）+vp” 实际上与 “介词 + 谓宾” 同理。

1****、根据后接成分区别连词介词的局限

根据后接成分的性质来区分介词和连词的方法实际上来自印欧语法。以英语为例，介词的后接成分只能是名词性的宾语，如果后接从句，则应该将该词归为连词。这一规则成立的前提条件是英语具有严格的形态变化，有着定式动词和不定式动词的区分。当一个兼类词连接一个带有动词的结构时，可以通过动词是否具有不定式动词的形式来判断这个结构是从句还是名词性宾语。但是汉语不具有形态变化，所以当一个兼类词连接含有谓词性成分的结构时，无法从形式上判断该结构是作为宾语还是作为分句存在的，主流的观点将这种情况直接归入连词类，是值得商榷的。

2****、“因为” 后接 “指称成分”

朱德熙在《语法讲义》中用指称和陈述来说明体词性成分和谓词性成分在特定句法位置上的语义功能。汉语中的动词和形容词可以直接充任主宾语而无需形态变化，但是在语义上有已经事物化和没有事物化的区别。具有指称功能的成分可以用 “什么” 指代，具有陈述功能的成分可以用 “怎么样” 指代。[⑤] 章欣基于此，认为 “位于‘因为’之后的谓词性成分全部已经指称化了，并且其意义跟谓词性结构的原因有关，都表示转指意义。”[⑥] 由此，“因为” 的三种结构皆能被 “因为 + 指称成分” 容纳。

朱德熙已经指出，汉语中词类和句法成分没有一一对应的关系，并且以陈述和指称来划定词的语义功能。沈家煊的 “名动包含” 说则更进一步，说明汉语陈述语用作指称语是 “构成关系” 而非 “实现关系”，汉语的陈述语是指称语的一个次类，汉语动词是名词的一个次类。[⑦] 那么，不论是从语义还是词类功能的角度，谓词充当宾语的合法性已经显而易见，“因为” 后接的谓词性成分，是可以作为介词宾语而存在的。

*3***、“因为 +** 谓词性成分” 能够成立

汉语词类与句子成分没有一一对应的关系，谓词性成分可以充当谓语、主语、宾语多种句子成分。朱德熙在《语法讲义》中提出体宾动词和谓宾动词的概念，打破了传统语法中认为动词不能带谓宾的观点。介词由动词虚化而来，应当具有与动词相似的语法功能。那么介词是否能够容纳谓宾，具有和动词相似的语法功能？

学者们已经对 “介词 + 谓宾” 结构进行了一定的研究。马杜娟分析出包含 “[对 + 谓词性成分]+ 谓词性成分” 的语块 [⑧]。蒋瑾媛考察了 “通过”、“经过”、“自从”“关于” 等介词，能够和动宾短语、主谓短语组合，从而论证 “为了” 只有介词词性 [⑨]。陈昌来的《介词与介引功能》将介词分为两类：只介引体词性宾语的介词和也能够介引谓词性宾语的介词 [⑩]。由此可证 “介词 + 谓宾” 是具有合理性的。

综上所述，因为 “动词 + vp”、“介词 + vp” 的结构能够成立，而 “连词 + vp” 的结构只能看成省略的特殊情况，所以 “因为 + vp” 结构中的 “因为” 如果被视为介词，会更加具有语法系统上的规整性。所以，本文更倾向于将 “因为 +（np+）+vp” 结构分析为介宾短语。

三、单复句划分与 “因为” 词类

“因为（+np）+vp” 结构中 “因为” 的词类划分问题，还受到学者们对于汉语单复句划分的认识的影响。若 “因为” 为连词，则承认以谓词结构划分单复句，若为介词，则不认可谓词结构独立成一分句。对于单复句的划分问题，学界尚存在一些分歧。综合来看，学者们通过小句结构上的互相独立、小句间的语音停顿、连词的使用、意义的关联等标准对单复句进行划分 [11]，但是仍然存在许多单复句界限不明的情况。刑福义指出了单句与复句在界限上互有瓜葛的诸多情况 [12]。孙良朋则主张汉语句子不再分单、复句，应该按照句法结构、语义关系对所谓 “复句” 进行分析 [13]。汉语单复句的划分问题，在古代汉语语法研究中犹为突出，刘利曾指出句子边界的不清以及复句与紧缩、连动、介词结构、代词复指结构的纠缠，都会导致单复句划分的不明 [14]。

古代汉语中，“分句” 这一概念并不常用。《马氏文通》连字章有一重要概念 “读”，在语法分布上，与现代汉语中偏正复句中的偏句相类似。然而，在现代汉语复句语法中，偏正复句的偏句和复句都是小句，在语法单位层级体系中的地位相同。然而，“读” 虽然可以具有起词和语词这样与 “句” 类似的主谓结构，但是马建忠并未给予 “读” 以 “句” 的同等地位，“读” 实际上是在核心结构之外的一个谓词性结构，在句中作一个指称性的成分。在古人的汉语实践中并未有单复句的观念，他们往往使用一些标记关系的虚词（有关联作用的副词、连词、介词）来表示逻辑语义关系。所以，马氏实际上是根据汉语的特点，以 “读” 代替了小句这一级语法单位。《文通》中的连字章也并不等同于现代意义上的 “连词”，而是将具有关联作用的虚词都吸纳进来，大约等同于 “关联词”。

但是，现代汉语相对于古代汉语发展出了更多语法范畴和形式手段，在一个句子中，一个主谓结构需要加上一些结构成分，才能构成一个意义完整的句子。例如 “夫子哂之” 可以成立，而 “夫子笑他” 就不太能说。在 “因为” 连接的结构中，有许多比单纯的主谓结构中更为复杂的形式。如果说 “因为 + np+vp” 结构大致能被介词结构容纳，那么当 “因为” 的后接部分包含了表示语气范畴、否定范畴、情态范畴、意愿范畴、时体范畴等完句成分时 [15]，就不能称之为动宾或主谓短语，只能称之为分句了。所以，主张取消复句的观点，在现代汉语研究中就显得执其一端了。

①多元化总是优于单一化，因为多元化包含了各单一化的优点，而避免了它的缺点。
②有的人因为他们不正确地忽视或排斥教育为社会需要服务的职能就予以全盘否定。
③原则上，他并不鼓励学生读历史，因为历史系毕业后出路很窄。

以上三句，“因为” 的后接成分都不是简单的主谓结构。在形式上，有几点与主谓短语相区别的特点：其一，具有语气范畴、否定范畴、程度范畴等完句成分；其二，结构内部可能有多个主谓结构，语气有停顿，结构相对复杂，本身就可以是复句；其三，该结构可以后置，并不出现在介宾短语的典型位置。由此我们可以判断，不同于 “因为（+np）+vp” 简单结构的模棱两可，这些结构中的 “因为” 已经完全成为连词了。

四、结论

在 “因为” 词类的探讨过程中，本文注意到以下两个方面的问题：其一，至少在共时层面上，基于汉语特点，我们能对 “因为（+np）+vp” 结构中 “因为” 词性进行重新探讨，将之归于介词和连词的过渡状态。其二，通过对古代汉语和现代汉语中复句的划分操作中的区别进行讨论，可以明确的是，从一些形式特征判断，一部分的 “因为” 已经从介词转化成连词。由此，我们需要在三类 “因为” 的语法结构之外另立一类 “因为 + 句”，尽管这一类的主干仍是主谓结构，但已经具有句子的特点。

本文最后得出的结论是，“因为” 的词类是在一连续统上进行变化的。“因为 + np” 是典型的介宾结构。“因为 + vp” 由于 “介词 + 谓宾” 结构的合法性可以将 “因为” 看作是介词，但是已经开始向连词转化。“因为 + np+vp” 已经具有了主谓结构，但仍然是一指称性成分，处于介词和连词的过渡地带。而当 “因为” 后接的主谓结构具有了句子的特点（完句范畴、语调、结构复杂度等），它已经不能为介宾结构所容纳，那么 “因为” 就已经完全成为了连词。
